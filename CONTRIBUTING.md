## How to Contribute

### 1. Find Work

Check the Issues tab or look at the task list above. Pick something you can deliver.

### 2. Do the Work

```bash
# Fork and clone
git clone https://github.com/YOUR_USERNAME/hivemind-sdk
cd hivemind-sdk

# Install deps
npm install

# Make changes
# ... code ...

# Test
npm test

# Build
npm run build
```

### 3. Submit PR

Create a Pull Request with:
- Clear description of what you built
- Your wallet address for payment
- Tests for new functionality

### 4. Get Paid

Once merged:
1. Project creator records your contribution percentage
2. When project completes, call `claimRewards(projectId)` with your wallet

- TypeScript strict mode
- ESLint for linting
- Vitest for testing
- Use async/await (no raw promises)
- Document public APIs with JSDoc
